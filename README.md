# CS50P Week 0 - Functions, Variables

## Getting Started

1. Click the green **Code** button and select **Open with Codespaces**
2. Once your codespace has loaded, close any Welcome tabs that might have opened by default
3. Execute `update50` in your codespace's terminal window to ensure your codespace is up-to-date; if prompted, click **Rebuild now**

# CS50 Assignment Template

This template provides a CS50-configured Codespace environment for both **CS50** (C programming) and **CS50P** (Python programming) courses.

## For Students
Click the green "Code" button and select "Open with Codespaces" to start working on your assignment.

## For Instructors
Use this template to create assignment repositories via GitHub Classroom. Replace this README with assignment-specific instructions.

## Environment
This template uses the official CS50 Codespace Docker image with pre-configured tools including:
- **C**: clang, make, valgrind, debug50, check50, submit50
- **Python**: python3, pip, cs50 library, check50, submit50
- **Other**: SQLite, Flask, and additional CS50 utilities
